🐾 Aula: Objetos em JavaScript

Olá, pessoal! Hoje vamos aprender o que é um objeto em JavaScript usando um exemplo bem simples: um gato! 😺

📦 O que é um Objeto?

Um objeto é uma forma de guardar informações e ações sobre alguma coisa.

Por exemplo, o gato tem:

  • Propriedades (como nome e cor)
  • Métodos (ações, como miar)

🧪 Exemplo de Objeto


var gato = {
  nome: "Frajola",
  cor: "preto e branco",
  miar: function() {
    alert("Miau!")
  }
}

🔍 Como usar?

Para ver o nome do gato:

console.log(gato.nome);

Para fazer o gato miar:

gato.miar();

Simples assim! 🐱

🎯 Atividade no p5.js

Agora é a sua vez! Acesse o site editor.p5js.org e crie um objeto com o tema que você quiser (um animal, uma fruta, um personagem...)

📌 O que o objeto deve ter:

  • 2 propriedades (como nome, cor, tamanho...)
  • 1 método que mostre um alert() ou console.log() com uma frase

💡 Exemplo básico:


var cachorro = {
  nome: "Rex",
  idade: 3,
  latir: function() {
    alert("Au au!")
  }
}

Depois de criar, chame o método com:

cachorro.latir();
